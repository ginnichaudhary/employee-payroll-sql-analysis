# Employee Payroll & Performance Analysis (SQL)

## Overview
A SQL project built using MySQL to analyze employee payroll, attendance, and performance.
The project demonstrates real-world HR analytics using multi-table queries.

## Database Schema
- employees
- attendance
- performance

## Key Analysis
- Department-wise average salary
- Salary categorization using CASE
- Attendance summary
- Performance-based employee insights

## SQL Concepts Used
- CREATE, INSERT
- WHERE, GROUP BY
- JOINs
- CASE statements
- Aggregate functions (AVG, COUNT)

## Tools
- MySQL Server
- MySQL Workbench

## Outputs
CSV reports exported from MySQL Workbench for analysis and reporting.
